# 21. 6021 : [기초-입출력] 단어 1개 입력받아 나누어 출력하기(설명)(py)
## 21.1. 문제 설명

알파벳과 숫자로 이루어진 단어 1개가 입력된다.

입력받은 단어의 각 문자를 한 줄에 한 문자씩 분리해 출력한다.

예시

s = input()

print(s[0])

print(s[1])

...

참고

s[0] 은 첫 번째 문자를 의미한다.

## 21.2. 입력
5개의 문자로 이루어진 단어 1개가 입력된다.

Hello

## 21.3. 출력
각 문자를 한 줄에 한 문자씩 줄을 바꿔 출력한다.

H<br>
e<br>
l<br>
l<br>
o
