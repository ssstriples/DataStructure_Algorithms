# 36. 6036 : [기초-산술연산] 단어 여러 번 출력하기(설명)(py)
## 36.1. 문제 설명
```
단어와 반복 횟수를 입력받아 여러 번 출력해보자.

예시

w, n = input().split()

print(w*int(n))

참고

문자열 * 정수 또는 정수 * 문자열은 그 문자열을 여러 번 반복한 
문자열을 만들어 준다.
```
## 36.2. 입력
```
단어와 반복 횟수가 공백으로 구분되어 입력된다.

love 3
```
## 36.3. 출력
```
입력된 단어를 입력된 횟수만큼 반복해 출력한다.

lovelovelove
```
