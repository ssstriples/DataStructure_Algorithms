# 69. 6069 : [기초-조건/선택실행구조] 평가 입력받아 다르게 출력하기(py)
## 69.1. 문제 설명
```
평가를 문자(A, B, C, D, ...)로 입력받아 내용을 다르게 출력해보자.

평가 내용
평가 : 내용
A : best!!!
B : good!!
C : run!
D : slowly~
나머지 문자들 : what?
```
## 69.2. 입력
```
영문자 1개가 입력된다.
(A, B, C, D 등 문자 1개가 입력된다.)

A
```
## 69.3. 출력
```
문자에 따라 다른 내용이 출력된다.

best!!!
```
